# NetPractice — TCP/IP addressing puzzles

Solo **web-based** exercise series (typically **no git repo**). Configure small networks until all machines can communicate.

## Project theme

Learn **IPv4 addressing**, **subnet masks**, **routing**, and **default gateways** through 10 progressive levels. No coding — pure networking literacy.

## How it works on campus

1. Download your personal exercise file from the **intra** (42 intranet)
2. Open the NetPractice web interface
3. Complete levels 1–10; progress is saved in your file
4. Evaluation: explain a random level to a peer

## Prerequisites

- Basic binary ↔ decimal conversion
- Understanding of IP dotted notation (`192.168.1.1`)
