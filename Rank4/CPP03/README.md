# CPP03 — C++ Module 03

**Inheritance**, **polymorphism basics**, and the **diamond problem** — ClapTrap family and virtual inheritance.

## Module theme

Build a small hierarchy (`ClapTrap` → `ScavTrap` / `FragTrap` → `DiamondTrap`) while managing constructors, energy/HP, and attack damage rules from the subject.
