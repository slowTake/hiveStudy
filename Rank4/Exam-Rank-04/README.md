# Exam Rank 04

Timed practical exam covering **C** fundamentals and early **C++** from Rank 4. Pass to unlock Rank 5 projects.

## Exam format

| Level | Topic | Example exercises |
|-------|--------|-------------------|
| 00 | Basic C, argv, write | `aff_first_param`, `ft_putstr`, `ft_strlen` |
| 01 | Strings, loops | `ft_strcpy`, `ft_swap`, `repeat_alpha` |
| 02 | String manipulation | `inter`, `union`, `wdmatch` |
| 03 | Memory, pointers | `ft_strdup`, `ft_range`, `fprime` |
| 04 | Linked lists, function pointers | `ft_list_size`, `ft_list_foreach` |
| 05 | Advanced C | `ft_itoa`, `ft_itoa_base`, `brainfuck` |

You receive **one random exercise per level**. Complete level 00 to get 01, etc. Typical pass: reach level 04 or 05 within the time limit — **confirm current rules on your campus**.
