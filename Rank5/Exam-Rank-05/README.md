# Exam Rank 05

Timed practical exam covering C++ OOP topics from Rank 5 (operators, inheritance, polymorphism). Level 00 exercises are typically **vect2**, **polyset**, and **bigint**; level 01 may include **life** (Game of Life).

This directory contains **study guides and implementation walkthroughs** — not copy-paste exam submissions. Write your own code from the algorithms described here.

## Exam format (typical)

| Level | Exercises | Difficulty |
|-------|-----------|------------|
| 00 | `vect2`, `polyset`, `bigint` (one random) | Operator overloading, OOP |
| 01 | `life`, `bsq`, … | Harder algorithmic |

You must pass level 00 to reach level 01. Practice all three level-00 exercises — you do not choose which one you get.

## Practice environment

Use the official exam shell (`exam` command) or a community practice repo that provides the subject `main.cpp` and expected file names. Always test against the **given main** before considering an exercise done.
